--- layout: post title: win nodejs下 findit只返回第一个文件问题 ---

目前截至本文的win nodejs v0.6.10下，findit依旧存在这个问题

参考https://github.com/substack/node-findit/issues/5，目前解决方案。

修改findit的index.js中的createInodeChecker函数，修改为

function createInodeChecker() {    var inodes = {};    return function inodeSeen(inode) {        if (inode == 0) {            return false;        }        if (inodes[inode]) {            return true;        } else {            inodes[inode] = true;            return false;        }    }}
