关于Git和GitHub.com的介绍忽略。本文假设按照要求该安装的都已经安装。

A. 首先登录github.com，创建一个新的Repository，假设名叫next.vifii.com

B. 如果本地项目没有使用Git的话，首先进入项目的目录，然后执行：

git config --global user.name “Weixiao Fan”git config --global user.email “i@vifiii.com”git initgit add . //添加所有文件git commit –m “first commit”git remote add origin git@github.com:fanweixiao/next.vifii.com.git //fanweixiao是我在github.com的用户名，next.vifii.com是我创建的repository的名字

C. ssh-keygen –C “i@vifiii.com” –t rsa

然后转到相应的目录下打开d_rsa.pub文件，将内容复制。打开github.com上创建的repository，点旁边的edit，在Deploy Keys里点add another deploy key，名字随便起，把刚才复制的内容粘贴在上面。

D. 然后输入SSH –v git@github.com可以检查

E. 运行git push origin master将本地库的内容同步到github.com上

至此，这个过程就ok了。
