--- layout: post title: rails插件的安装 ---

通常情况下、莪们可以使用命令行来安装、比如下面的例子、

Install a plugin:rails plugin install continuous_builder Install a plugin from a subversion URL:rails plugin install http://dev.rubyonrails.com/svn/rails/plugins/continuous_builderInstall a plugin from a git URL:rails plugin install git://github.com/SomeGuy/my_awesome_plugin.gitInstall a plugin and add a svn:externals entry to vendor/pluginsrails plugin install -x continuous_builder
