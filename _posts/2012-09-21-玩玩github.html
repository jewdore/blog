安装部分：

安装这东西比我想象中的简单多了，找了好多中文的博客，但是发现官网的指导才是最好的入门指导，虽然是英文但是超简单的讲解也特详细

官网帮助：http://help.github.com/win-set-up-git/

网友的博客：http://www.kuqin.com/managetool/20111113/314830.html

我按照官网的指示一路下来很顺利的，大体思路就是：

1.官网注册账号

2.安装客户单（downloadUrl：http://code.google.com/p/msysgit/downloads/list）

3.客户端与服务器端的绑定

4.验证是否成功。

--------------------------------------------------------

正式开始使用github：

1.首先在网站上新建一个repository，进入你的账号，点击new repository

2.填写后面的三项，当然最后两项可以不填Project Name (optional) (optional)

3.打开Git Bash

mkdir ~/MyFirstProjectTest (~指的是该环境的根目录，后面的是在github上建立的工程的名字)

cd MyFirstProjectTest

git init

touch README

git add README

git commit -m 'my first commit'

git remote add origin git@github.com:shanshangzhangran/MyFirstProjectTest.git

git push -u origin master

这个过程中你需要输入自己的passphrase（口令）

这样就成功的创建并提交了自己的README 了解破碎机常识：crusher mill supplier
