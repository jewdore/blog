版本控制管理、现在比较流行的是SVN、目前又多了个git、也就是分布式、具体怎么区别也就不介绍了、

现在说说github的使用、

安装git后、

如果本地主干和github的主干不一样、必须先使用pull命令拉取github的文件至本地、然后再回本地执行add、commit、push提交至github合并主干、记住必须包含master、

$ git pull git_code_remote master$ git commit -m "update code."$ git push git_code_remote
