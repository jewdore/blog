作为vim用户，xcode真是让我抓耳挠腮，难受的要死，～～咬牙切齿还是把MacVim给装上，可是光有个vim不顶用啊。我一直使用exvim这个套装（神器，没用过的赶紧上），要一堆unix上的开源玩意，一个一个弄，我还不给它搞死。liunx上apt-get那玩意没道理mac上没有替代品，一阵google，果然有，还不知一个。作为软件试用深度上瘾症患者，赶紧的一个一个试。

Brew（homebrew主页）：

好，就一个字评价，如果要安装wget，只要这样：

 brew install wget

brew会自动安装需要的依赖，然后symlinked into /usr/local。

卸载也简单：

brew uninstall wget

ok, 赶紧的装brew吧，用下面这个命令，一键安装

curl-LsSfhttp://github.com/mxcl/homebrew/tarball/master|sudotarxvz-C/usr/local--strip1

缺点也是比较明显，能安装的软件比较少，像我要安装的sed，id-utilts都没有，哎～～～，希望以后会多起来吧。

fink:

是基于apt的，软件比起brew多一些，但也不是很全，它的目标是把全套的unix工具挪到mac上，希望在远方了.. 还有个问题就是软件版本比较老，哎，应该是活跃度比较低，玩的人比较少的缘故。

安装fink：

就没有brew那么容易了，10.6还没有二进制安装包，只能从source上编译，前提是你装了xcode (非程序员别整了）。

安装指南在这里：

http://www.finkproject.org/download/srcdist.php 一步一步来就是了，

装好后：

fink install sed -》 搞定， 果然软件比起brew是多点

tip:

万一那个软件没有，你就执行下fink update-all 碰碰运气 :-)

MacPorts

最老牌的东西，活跃度最高，软件最多。自身的安装友好，提供全界面的安装。缺点么，也是不太清楚，网上不少同学对它相当感冒。

管他了，装了再说，是好是歹，溜久了就知道了。

命令也是一样的:

port install sed

port update sed

port uninstall sed
