Code Cook？

　　偶尔看《计算机程序设计艺术 第三卷》，卷首引用了一句话，“Cookery is become an art，a noble science；cooks are gentleman.”，烹调是一种艺术，而作者认为编程同样是一种艺术，遂名曰“Code Cook”。

现况描述

　　经过半年的努力，现已进入武汉大学计算机学院继续学习，在复习考研的过程中，有过很多思考，关于考研的一些总结，记录在这个帖子上，通过写这个帖子，发现原来写东西真不是一件很容易的事情；今后一些想法和学习所得会记录在这个博客，以便交流。

　　“书写是为了更好的思考”[1]，书写的同时可以锻炼自己的思维和表述能力，写下自己的想法和心得，可以更好的听取大家的意见，从而自我反思，引发进一步的思考。

　　此刻，笔者正在武汉大学珞珈图腾数据库实验室专利小组学习，该小组主要负责一个专利检索和统计分析系统的开发和维护，使用自主开发的数据库Totem进行数据存储。

未来规划

　　研究生期间想要做的事情很多，归根结底还是想将来能找到一个自己感兴趣的工作，实现人生的价值。

　概括的讲，主要做以下几件事情：

读书，相关书单会在之后的博客中详细阐述。博客，坚持写自己的文章，广泛的阅读，独立的思考。Github，参与一些开源项目的编写。英语，自知英语学习的不是很好，强迫自己翻译一些专业文献。

　　　　

　　You just need a little faith，believe inyourself。

[1]： 刘未鹏大牛说过的话，很有道理，他的博客很值得关注。
