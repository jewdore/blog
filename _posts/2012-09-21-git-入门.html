二、日常使用：

Global setup:   Set up git   git config --global user.name "yulike"   git config --global user.email likeyu@gmail.com       Next steps:   mkdir blardrails   cd blardrails   git init   touch README   git add README   git commit -m 'first commit'   git remote add origin git@github.com:yulk/blardrails.git   git push -u origin master       Existing Git Repo?   cd existing_git_repo   git add .   git commit -m 'full commit'    git remote add origin git@github.com:yulk/blardrails.git   git push -u origin master      

另外git add –i 是一个交换方式，可以选择要更新的文件。http://imtx.me/archives/1297.html
