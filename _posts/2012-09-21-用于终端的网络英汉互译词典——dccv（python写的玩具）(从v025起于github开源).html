--- layout: post title: 用于终端的网络英汉互译词典——DCCV（python写的玩具）(从V0.25起于github开源) ---

某天突然装了sdcv和dict发现没有英-汉的本地词库，也懒得装了，于是自己用python写了个基于dict.cn的网络版

就叫Dict.Cn Console Version了

有需要的朋友随意拿去用

写得很随意，欢迎赐教

changelog:

v0.25 添加Cygwin平台的剪切板支持，于github开源v0.24修正汉英查询时的编码问题v0.23 添加输出着色，优化排版（来自tan_Cool的建议）v0.22 添加安装成功后的用法提示；修复无法找到空白缓存文件的bug（感谢tan_Cool的热心提示和patch）v0.20 添加缓存功能，每次查询的单词将会缓存到本地，下次查询同单词无需联网（感谢DeathSpank的建议）v0.11 完善了install.sh，下了老版本的朋友请运行一次“chmod +x /path/to/dccv.py”否则有可能无法使用d代替dcvv.py)

暂只在Linux下测试通过，MAC和WIN下应该有更好的图形化替代品

下载后运行install.sh即可安装

安装后可直接使用d代替/path/to/dccv.py，使用方法如下图

注意：安装后不要更改dccv.py所在路径

若已安装旧版无需再次运行install.sh，下载新版覆盖掉同名文件即可

卸载方法：

1.删除dccv目录2.编辑文件"~/.bashrc"，找到并删除从
#Start alias for dccv
到
#End alias for dccv
的三行，保存即可
v0.22之前的版本请找到并删除一下行
alias d=/path/to/dccv.py
