--- layout: post title: 在Linux中使用Git笔记 ---

在Linux下搞开发，不用Git来进行版本控制是有些浪费，但是以前都是用SVN，而且都是图形界面，用这个显然入手不太容易，今天整了挺久，总结一下。

1、在https://github.com上申请号先

我是要将代码放到github中的，如果你不想放，那可以掠过这步直接看后面的，因为Git支持本地和服务器上的Repository。

申请好后，先创建一个新的Repository，记好自己的帐号，邮箱，以及Repository名称就好。

例如我创建了一个帐号test，邮箱test@test.com，Repository为testRep

2、在linux下安装git

我的系统是Ubuntu12.04，所以直接安装搞定

sudo apt-get install git git-core
