--- layout: post title: Windows下安装node.js ---

我是参考这位博友的安装方式http://www.cnblogs.com/seanlv/archive/2011/11/22/2258716.html

第一步，下载Node.js的windows安装程序和Npm

地址如下：

node.jshttp://nodejs.org/#download

npm　　https://github.com/isaacs/npm/tags

第二步，安装node.js，把安装目录下的node.exe拷贝到自己新建的D:/nodejs/目录下，

然后添加Path环境变量D:/nodejs/，然后新建一个D:/npmjs/目录将npm压缩包加压到里面，

添加环境变量NODE_PATH值为D:\nodejs\node_modules

第三步,执行以下命令

D:\>cd npmjs

D:\npmjs>node cli.js install -gf

进行安装

然后安装express

D:\npmjs>npm install express -g

这个是从Internet下载

到这里面就算环境安装完毕

第四步，测试

用node -v，npm -v，express -v 来检测其安装是否成功

创建测试项目

D:\>express hello

D:\>cd hello

D:\hello>npm install （加入依赖项）

D:\hello>node app.js(启动，会提示启动端口）

在浏览器中输入http://localhost:port即完成node.js的安装

说明：

系统为XP SP3，

express为2.5.8

npm为1.1.16

node为v0.6.14
