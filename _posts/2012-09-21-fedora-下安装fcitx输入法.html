--- layout: post title: Fedora 下安装Fcitx输入法 ---

默认情况下Fedora的库里面是只有fcitx 3.6.3 输入法的，在已经出现4.0.X版本的时候，想要装新版fcitx就要用些手段了

到fcitx的googlecode项目主页上可以看：

动态

2010-12-17fcitx 更新 4.0.1 fcitx-configtool 更新 0.2.0 fcitx-sunpinyin 更新 0.2.1

2010-11-28fcitx-sunpinyin 更新 0.2.0

2010-11-21版本管理改为 mercurial，将 fcitx-config 和 fcitx-sunpinyin 从 github 也迁移到了 google code。

2010-11-19发布 3.6.4，4.0.0

2010-11-09发布 4.0 rc1

2010-11-02删除了 svn 里的原本打算用来存储码表的 /extra 目录，并将被删除的一个码表压缩包移到了下载列表里。在 svn 里存储压缩包等二进制文件不是一个好的实践。

2010-10-12增加了 openSUSE和Fedora的源，地址见右侧外部链接（External Links）(我们需要的就是这句话)

2010-09-23增加 PPA 源，（External Links）Google code 上的 deb 包将不再更新

1）于是按照指示可以找到 home:csslayer.repo

2）下载下来放到 /etc/yum.repos.d里面

3）yum makecache

4）sudo yum install fcitx （当然最好能够先把自带的IBUS输入法删除掉 sudo yum remove ibus）

5）sudo vi /etc/profile 或 sudo vi ～/.bashrc

在里面加上：

export XMODIFIERS="@im=fcitx"

export QT_IM_MODULE=xim

export GTK_IM_MODULE=xim

这样就可以安装最新版本的输入法了

安装完成后要注销一下系统

如果进入系统时Fcitx输入法不能自动启动，点击系统-->首选项-->启动应用程序 对话框中点击添加 名称 命令位置分别填上 fcitx fcitx 即可

安装词库

到 http://code.google.com/p/fcitx/downloads/list 下载词库，本人用的是“搜狗细胞词库，hubert_star修改（utf-8）”,下载后解压，把解压后的pyphrase.mb复制到/usr/share/fcitx/data 目录下
