下面摘自http://www.ibm.com/developerworks/cn/linux/l-django/#N100EF，对django的一段简介（该链接还提供了django的使用方法，蛮简单的）：

Django 是 Python 编程语言驱动的一个开源模型-视图-控制器（MVC）风格的 Web 应用程序框架。使用 Django，我们在几分钟之内就可以创建高品质、易维护、数据库驱动的应用程序。Django 项目是一个定制框架，它源自一个在线新闻 Web 站点，于 2005 年以开源的形式被释放出来。Django 框架的核心组件有：

（1）用于创建模型的对象关系映射

（2）为最终用户设计的完美管理界面

（3）一流的 URL 设计

（4）设计者友好的模板语言

（5）缓存系统
