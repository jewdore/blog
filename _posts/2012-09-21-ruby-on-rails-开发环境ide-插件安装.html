只需要安装下面几个插件即可 实现一个强大的支持语法高亮、自动补全、语法错误分析的ruby rails 开发环境

ctags http://www.vim.org/scripts/script.php?script_id=610

taglisthttp://www.vim.org/scripts/script.php?script_id=273

安装方法：

下载后解压缩，放入~/.vim/目录下即可。

特别：

1.ctags ,需要先安装配置好。

1.1 :http://prdownloads.sourceforge.net/ctags/ctags-5.8.tar.gz

1.2 tar 解压缩，./configure ， make ,make install

1.3 vim 插件ctags http://www.vim.org/scripts/script.php?script_id=610

2.Taglist安装（依赖于ctags）

　　1、下载一个Taglist的zip文件，然后解压缩，将taglist.vim复制到~/.vim/plugin目录下。

　　2、修改~/.vim/plugin/taglist.vim

　　在 if !exists('loaded_taglist')上面加入

　　let Tlist_Ctags_Cmd="/usr/bin/ctags" //path 可能不一样，具体路径 可以用 ： which ctags 命令 获得。

具体见 每个插件压缩包的 Readme

ok！ 强大的支持语法高亮、自动补全、语法错误分析的ruby rails 开发环境 就差不多搭建好了。
