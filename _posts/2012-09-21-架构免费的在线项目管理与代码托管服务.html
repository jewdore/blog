--- layout: post title: 架构免费的在线项目管理与代码托管服务 --- 项目管理：

heroku.com 是除了 ms,google 这些金主以外数一数二的云平台，提供了免费的 ruby on rails 运行环境，无限应用个数，每个应用有 100mb 硬盘空间，5mb PostgreSQL 数据库（寒碜了点，不过对于小团队基本够用了）。heroku.com 的免费空间可以完美地运行 Redmine 项目管理系统，除了需要对 Redmine 源码做少许修改之外，Redmine 不会有任何功能缺失。

与 hostedredmine.com 之类的免费 Redmine 托管服务不同， 在 heroku.com 提供的云主机中你自己能够完全掌握 Redmine 的源代码，可以做自定义的修改。更重要的是，你的 Redmine 用的数据库是完全独立的，通过 heroku.com 提供的服务可以将其完全备份出来。如果你某天不想用 heroku.com 或者资源超限了不想缴费，那么可以直接把你的应用和数据完全移植到本地。

heroku.com 上 Redmine 实例请参考我们的项目系统：wychmine.heroku.com。

heroku.com 提供了免费的域名绑定和 SSL 支持，虽然是免费的，但如果要启用这些附加 Addons 需要进行信用卡验证。

代码版本管理：

github.com 是目前最流行的 Git 代码托管服务，但 github.com 的免费套餐只支持开源项目，不提供私有仓库方案。

作为 github.com 的强力竞争者，bitbucket.org 提供了无限数量的私有 Git 仓库，唯一的限制是最多只能为私有 Git 仓库指定 5 位访问用户。

bitbucket.org 身后的公司就是 JIRA 等若干工具的开发商，对于服务的持续性和稳定性是很好的保障。

bitbucket.org 的代码仓库也同时提供 wiki、bug 跟踪功能，但没有 Redmine 成熟和好用。
