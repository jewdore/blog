Grails基于groovy语言，是动态语言的快速开发框架，集成了JAVA的成熟框架。

Grails is an open source web development framework that packagesbest practices such as convention over configuration and unit testing with the best-of-the-best open source application frameworks such as Spring, Hibernate, and SiteMesh.Together with the productivity of the Groovy scripting language, everything runs on topof the robust Java and Java EE platforms.

无法上网环境如何建立环境

把需要的插件 .zip 文件放到C:\Java\grails-1.3.7\plugins下

把插件依赖的jar包放在C:\Java\grails-1.3.7\lib下（每个插件的dependencies.groovy中有依赖的内容）

如果插件很多也很麻烦

在能上网的机器上安装相同的版本的grails，路径也一致，然后建立一个项目安装所有需要的插件，完成后，在C:\Users\admin\.ivy2\cache中会保留所有插件和jar的缓存，把这个目录拷贝到另一个机器上，这个项目的运行就不用再下载插件了
