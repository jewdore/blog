--- layout: post title: Windows XP下用git同步Vim配置文件和插件 ---

1.创建git仓库，用于存放vim的配置文件_vimrc和插件以及插件管理插件pathogen.vim（以fugitive.vim插件的安装为例）。

cd C:\Program Files\Vim

copy _vimrc vimfiles

git init

git add .

git commit -m "initial vimrc"

2.创建_vimrc文件的硬链接

fsutil hardlink create "C:\Program Files\Vim\_vimrc" "C:\Program Files\Vim\vimfiles\_vimrc"

3.添加插件管理插件pathogen.vim

从http://www.vim.org/scripts/script.php?script_id=2332下载后

cd C:\Program Files\Vim\vimfiles

md autoload

copy pathogen.vim autoload

并在_vimrc文件中添加下面几行文字

call pathogen#runtime_append_all_bundles() 了解破碎机常识：crusher mill supplier
